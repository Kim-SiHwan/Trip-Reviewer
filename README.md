# TripReviewer
## 소개
TripReviewer는 대한민국 행정구역 데이터를 시각화 해 
자신만의 지도를 생성하고, 방문 표시와 사진첩에 사진 삽입 등
가꾸며 여행지의 리뷰를 통한 다른 사람들과의 소통을 할 수 있는
프로젝트입니다.
